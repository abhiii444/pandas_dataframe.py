# pandas_dataframe.py
This project is a Pandas DataFrame Exploration Script that demonstrates how to create, manipulate, index, slice, filter, modify, and analyze DataFrames using the Python Pandas library. It includes examples of DataFrame creation, column/row selection using loc and iloc, conditional filtering, adding/removing columns, performing operations.
# Pandas DataFrame Operations & Exploration
